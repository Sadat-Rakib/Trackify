## 🚀 Quick Start

```bash
# Clone your wellness future
git clone https://github.com/Sadat-Rakib/Trackify.git
cd Trackify

# Install dependencies
npm install

# Start development server
npm run dev

# Visit: http://localhost:3000
```

5. **Start Tracking** - Begin your transformation journey
